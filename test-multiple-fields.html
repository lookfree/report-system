✅ 实现效果

  • 支持在单个单元格中插入多个数据集字段
  • 字段之间可以添加任意说明文字
  • 内联显示方式，与文本自然融合
  • 紫色渐变背景标识，带呼吸动画效果
  • 预览时自动替换为实际数据值
  • 支持单条数据和列表数据混合展示

使用说明

  1. 插入字段：双击表格单元格，选择"单条"模式，选择要插入的字段
  2. 添加文字：在字段占位符之间直接输入说明文字
  3. 继续插入：将光标放在需要的位置，再次双击插入新字段
  4. 预览效果：点击预览按钮查看实际数据替换效果
